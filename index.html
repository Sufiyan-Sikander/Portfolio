Featured Projects

ShurukerAi-FYP

ShurukerAi is a full-stack AI-powered business intelligence and freelancing platform built with FastAPI and React.js. It combines Retrieval-Augmented Generation (RAG), real-time Firebase services, and vector search using Pinecone to deliver an intelligent ecosystem for startup insights, AI chat assistance, freelancer onboarding, and client–freelancer communication. The system enables users to interact with an AI business assistant, explore market trends across cities in Pakistan, evaluate business probabilities, and connect directly with verified freelancers through a structured marketplace. The backend is deployed on Hugging Face using FastAPI, while the frontend is deployed on Vercel for fast and scalable delivery.

  • FastAPI backend optimized for high-performance AI inference and API handling
  • React.js frontend with responsive UI and dynamic state management
  • LangChain-based RAG pipeline for document retrieval and response generation
  • Pinecone vector database for semantic search and context retrieval
  • Firebase (Auth, Firestore, Storage) for identity, chat, and data persistence
  • Deployed on Hugging Face (backend) and Vercel (frontend) for production scalability

It enables users to interact with an AI chatbot for traffic-related queries, challan calculations, driving license information, accident statistics, and real-time alerts. AI Content Summarizer

URL Content Summarizer is a lightweight AI-powered web application that extracts and summarizes content from YouTube videos and web URLs. It is built using Flask and LangChain with Groq's Llama-3.1-8B Instant model to generate high-quality structured summaries. The system supports multi-format outputs including detailed English summaries, key bullet points, Urdu translations, and timestamps for videos. It uses intelligent document loading and chunking to handle long-form content efficiently.

  • Flask backend for handling API requests and serving web interface
  • LangChain-based summarization pipeline using map-reduce strategy
  • Groq Llama-3.1-8B Instant model for fast and efficient inference
  • YouTube transcript extraction using YoutubeLoader
  • Web content extraction using UnstructuredURLLoader
  • Support for English summary, key points, and Urdu translation output

Text To Math Problem Solver

Text To Math is an AI-powered reasoning assistant that converts natural language questions into mathematical solutions. Built with Streamlit and LangChain, it uses Groq's Llama-3.1-8B Instant model combined with a ReAct agent to intelligently select between a calculator, Wikipedia search, and a reasoning chain to answer complex math and logic problems step by step.

  • LangChain ReAct agent with dynamic tool selection
  • Calculator tool using numexpr for safe numeric expression evaluation
  • Wikipedia search tool for factual context and knowledge retrieval
  • LLMChain-based reasoning tool for multi-step logic problems
  • Streamlit chat interface with persistent session history
  • Groq Llama-3.1-8B Instant for fast low-latency inference

Customer Churn Prediction App

Customer Churn Prediction is an AI-powered machine learning web application that predicts whether a bank customer is likely to leave the service. It is built using Streamlit for the frontend interface and TensorFlow/Keras for deep learning inference. The system takes user inputs such as geography, gender, age, balance, credit score, salary, and account activity, then preprocesses them using trained encoders and a scaler before feeding them into a neural network model.

  • Streamlit frontend for interactive real-time prediction UI
  • TensorFlow/Keras deep learning model for churn classification
  • One-hot encoding for categorical feature (Geography)
  • Label encoding for binary categorical feature (Gender)
  • StandardScaler for feature normalization
  • Real-time probability output with threshold-based classification logic
